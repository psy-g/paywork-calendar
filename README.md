## Build Installation 💻

```bash
# repository clone
$ git clone https://github.com/psy-g/paywork-calendar.git
# change directory
$ cd paywork-calendar
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run start
```
